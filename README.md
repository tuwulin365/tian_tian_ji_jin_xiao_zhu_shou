# 纳斯达克100基金限额监控Chrome扩展

## 功能说明
该Chrome扩展用于监控天天基金网纳斯达克100场外基金的每日申购限额变化，当限额发生变化时会及时通知用户。

## 支持的基金
- 161130: 易方达纳斯达克100发起式(QDII-ETF联接A)
- 006479: 国泰纳斯达克100(QDII)
- 000834: 广发纳斯达克100指数A(QDII)
- 005503: 华夏纳斯达克100ETF联接A
- 162411: 华宝标普油气上游股票人民币A

## 主要功能
1. **自动监控**：每天自动检查一次基金限额
2. **实时通知**：当限额发生变化时，通过Chrome通知提醒用户
3. **手动检查**：用户可以随时点击"立即检查"获取最新数据
4. **历史记录**：显示最近一次检查的时间和结果

## 安装方法
1. 下载本扩展的所有文件到本地文件夹
2. 打开Chrome浏览器，进入扩展管理页面（chrome://extensions/）
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"，选择下载的文件夹

## 使用说明
1. 安装完成后，点击Chrome工具栏中的扩展图标
2. 首次使用点击"立即检查"获取最新数据
3. 扩展会自动每天检查一次限额变化
4. 当限额变化时，会收到桌面通知

## 技术实现
- **Manifest V3**：使用最新的Chrome扩展标准
- **Service Worker**：后台定时任务管理
- **DOM解析**：从基金页面提取限额信息
- **本地存储**：保存历史数据和变化检测
- **通知系统**：限额变化时发送提醒

## 注意事项
- 本扩展仅提供信息查询功能，不构成投资建议
- 基金限额数据来源于天天基金网，仅供参考
- 请确保网络连接正常，否则可能无法获取最新数据
- 扩展需要网络权限和通知权限才能正常工作
